# 🏨 Hotel Reservation System  
## Python Programming Project
The Hotel Reservation System is a console-based Python application designed to simulate basic hotel management operations. It provides functionality for managing rooms, registering guests, creating and handling reservations, and performing check-in and check-out processes.
The project emphasizes the use of object-oriented programming (OOP), including inheritance and encapsulation, along with exception handling, file-based data storage, and modular program structure. It is intended as a practical assignment for students to apply core Python programming concepts in a real-world scenario.

---

## 📌 Project Overview  
You are required to develop a **console-based Hotel Reservation System** using Python.  

The system should simulate how a real hotel manages:
- rooms  
- guests  
- reservations  
- check-in and check-out processes  

---

## 🎯 Objectives  
By completing this project, you should be able to:
- design a program using **classes and objects**  
- manage data and relationships  
- handle errors and invalid input  
- store and retrieve data from files  
- build a structured application  

---

## 🧩 Functional Requirements  

### 1. Room Management  
- Add new rooms  
- Remove rooms  
- View all rooms  
- View available rooms only  
- Search for a room by number  
- Support different types of rooms  

---

### 2. Guest Management  
- Register new guests  
- View all guests  
- Search for a guest  

---

### 3. Reservation Management  
- Create a reservation  
- Cancel a reservation  
- View all reservations  
- Search for a reservation  

---

### 4. Check-in / Check-out  
- Check in a guest  
- Check out a guest  
- Prevent invalid operations  

---

### 5. Pricing and Stay Calculation  
- Calculate total cost based on stay duration  
- Handle dates correctly  
- Support different pricing rules for different room types  

---

### 6. Reports (at least 3 required)  
Examples:  
- Available rooms  
- Active reservations  
- Completed stays  
- Total income  

---

### 7. Data Persistence  
- Save data to files  
- Load data when the program starts  
- Ensure data is not lost between runs
